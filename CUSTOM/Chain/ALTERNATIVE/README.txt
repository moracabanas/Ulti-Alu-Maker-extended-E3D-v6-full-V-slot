# Summary

This is a remix of Robert's cable chain to fit <a href="http://www.thingiverse.com/jasonatepaint">jasonatepaint</a>'s <a href="http://www.thingiverse.com/thing:811271">Ultimaker 2 Aluminum Extrusion 3D printer</a>

Width 2mm narrow than original for all parts and the one connected to the bed is shorter. It only have 1mm clearance to the aluminum extrusion and about 2-3mm clearance to the bed



# Parts

1 x ChainBed.stl
?? x ChainSegment.stl
?? x ChainLid

Since my Um2 alu extrusion is Extended size I used 14 segments and lid, it may vary from your printer size.

# NOTE:

I am using the Heated bed upgrade from Ultimaker but it should fit the UM2 bed as well. My version is a Extended size so the chain are more than what a normal size UM2 would need
